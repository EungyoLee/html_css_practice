<table> 태그와 <input> 태그입니다!
table 태그는 표를 만드는 태그, input 태그는 입력을 받는 태그입니다.
type="text" 일반적인 텍스트를 입력받는 칸
type="password" 비밀번호 등을 입력받는 칸
type="submit" 누르면 서버로 데이터를 전송하는 버튼을 생성
type="reset" 누르면 작성한 모든 내용을 초기화하는 버튼을 생성
※ submit과 reset 버튼은 같은 form 태그 내부에 속한 input 태그들에만 적용된다.
type="radio"


여러가지 선택 중 하나를 선택할 수 있는 버튼
하나만 선택하기 위해서는 name 속성이 같아야 한다.
type="checkbox"

하는 역할은 radio 버튼과 비슷하지만 복수 선택을 할 수 있다.
type="email"

이메일을 입력하는 칸을 생성한다.
사용자가 입력한 내용이 이메일인지 검사한다.
type="color"
사용자가 색상을 선택할 수 있는 태그
값은 r, g, b값이 16진수 6자리로 전송된다.
type="date" 사용자가 날짜를 선택할 수 있게 해준다.
